Help:

The machine is viewing the library.

The name of books are displayed one per line - truncated if too long.

Short pressing the selection buttons beside the book titles open the corresponding book at the place last used.

Long pressing the selection buttons beside the book titles open the book menu for that book.

The last line displays the current and total number of pages in the library and which library is being used (internal or USB).

Short pressing the selection button beside the last line opens a quick access menu to navigate the library.

Long pressing the selection button beside the last line opens the library menu.

Short pressing the left and right navigation arrows move through the library listing (if more than one page).

Long pressing the left and right navigation arrows moves to the first and last library listing pages respectively.

Short pressing the home button does nothing (as you are already in the library).

Long pressing the home button opens the library menu.
