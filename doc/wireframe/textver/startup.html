Help:

The machine is starting up.

It displays a message on the display to say so whilst it is loading.

The selection and navigation buttons continue to be inactive.

Switching the rocker switch again will put the machine into shutdown mode.

Once it has loaded it will automatically return to state it was in when last shut down - either viewing the library or a book.
